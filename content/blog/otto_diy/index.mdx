特にプリントしにくい形状もなく、ラフトもサポートもなしでPLAでプリントしました。  
パーツによって違う色でプリントすると可愛くなったり、かっこよくなったりしますね。

## 充電

9Vの角型充電式電池がキットには付属していました。 
(初めてみました。これはいろいろ便利そうです)

電池のおしりにMicroUSB端子がついていてそこにUSBケーブルを繋いで充電します。  
Ottoを組み立てたあとも、分解などせずに充電できるようになっています。

充電中は赤、充電完了で緑のLEDの点灯が確認できます。

## プログラム環境

今回はChromebookで完結できるように工夫しました。

2つのツールを使用します。

1. Arduino web editor  
   Arduinoマイコンのプログラムを作成しマイコンに
   書き込むツール
1. Otto Blockly Web  
   Ottoのプログラムをブロックを組み合わせて簡単に作成するツール

どちらもWEBブラウザで動作するツールなので、Chromebook用というわけではなく、WEBブラウザーとインターネットアクセスできれば使えるはずです。

Otto Blockly Webだけで本来はArduinoマイコンに書き込みまでできるのですが、そのためにインストールするプログラムが必要で、そのプログラムのChromebook版がなかったので、書き込みはArduino web editorを使いました。

### Arduino web editor
Arduinoマイコンのプログラムを作成しマイコンに書き込むツールです。

詳しい使い方は以下の記事を参照しました。
<iframe title="Arduino　web editor" src="https://hatenablog-parts.com/embed?url=https://qiita.com/rune187/items/771857828106c8114de6" style="width:100%;height:150px; max-width:600px; margin-left:auto; margin-right:auto;" frameborder="0" scrolling="no" loading="lazy"></iframe>

### Otto Blockly Web
Ottoのプログラムをブロックを組み合わせて簡単に作成するツールです。  
ブロックをドラッグアンドドロップで繋げてプログラムを作成するビジュアルツールです。

Otto用の動作パターンや音などのブロックが用意されているので、簡単にOttoのプログラムができます。

さらに、Otto用のサンプルも用意されているので、それを使えばすぐにOttoが動かせます。

ただし、Chromebookの場合はOtto Blocky WebからOttoへプログラムの書き込みができません。  
なので、Blockly Webをコードモードにして生成されたプログラムコードをコピーし、それをArduino web editorに貼り付けて、Arduino web editorからOttoに書き込みを行います。
