# HACSとは？

Home Assistant Community Store (HACS)

Home Assistantの非公式な統合、プラグイン、UI要素などを管理するためのツールです。コミュニティ製の豊富なコンテンツを簡単に見つけ、インストール、そして更新できるため、Home Assistantの機能を大幅に拡張し、よりパーソナルなスマートホーム環境を構築するのに役立ちます。
